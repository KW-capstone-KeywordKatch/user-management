# 회원 관리 컴포넌트

[KeywordKatch](https://github.com/KW-capstone-KeywordKatch/index#keywordkatch)의 회원을 관리하는 컴포넌트

## 기능
* 회원가입, 로그인, 탈퇴
* 관심분야 설정
* editor 컴포넌트로 "사용자-관심분야" 정보 전송

## 기술 스택
* Application: `Java` `Spring Boot`
* Deployment: `AWS EC2` `Docker`
* Database: `MySQL` `AWS RDS`
